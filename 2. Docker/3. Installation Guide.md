# A Simple Docker Installation guide

- Docker can be installed on various operating systems such as Linux, Windows, and macOS.
-  In this guide, Docker is installed on an **Ubuntu-based AWS EC2 instance** using the Ubuntu package manager (`apt`).
- After installation, the Docker service is started, enabled to run automatically at boot, and verified by running a sample container to ensure Docker is working correctly.

## Steps:-

### 1. Create an AWS EC2 instance.
#### 2. SSH into that EC2 Instance 
`ssh -i <key.pem> ubuntu@<EC2-Public-IP>`
#### 3. Docker Installation

  - Update Packages : `sudo apt update`

  - Install Docker : `sudo apt install docker.io -y`

  - Check Docker version : `docker --version`

  - Start Docker Service : `sudo systemctl start docker`

  - Enable Docker at boot : `sudo systemctl enable docker`

  - Check Docker status : `sudo systemctl status docker`

  - Give Docker Permission to Current User : `sudo usermod -aG docker ubuntu`

#### 4. Test Docker Installation 
`docker run hello-world`
- If Docker is installed successfully, Docker downloads the **hello-world** image and displays a confirmation message.
