# Docker Commands
- Docker commands are used to build, run, manage, monitor, and troubleshoot Docker images, containers, networks, and volumes.

## 1. Docker Information

| Command | Description | Example |
|---------|-------------|---------|
| `docker --version` | Displays the installed Docker version. | `docker --version` |
| `docker version` | Displays client and server version information. | `docker version` |
| `docker info` | Displays Docker system information. | `docker info` |
| `docker system df` | Shows Docker disk usage. | `docker system df` |

---

## 2. Docker Images

| Command | Description | Example |
|---------|-------------|---------|
| `docker pull` | Downloads an image from Docker Hub. | `docker pull nginx` |
| `docker images` | Lists all local images. | `docker images` |
| `docker image ls` | Lists Docker images. | `docker image ls` |
| `docker build` | Builds an image from a Dockerfile. | `docker build -t myapp .` |
| `docker tag` | Creates a new tag for an image. | `docker tag myapp:v1 myrepo/myapp:v1` |
| `docker push` | Uploads an image to Docker Hub or a registry. | `docker push username/myapp:v1` |
| `docker rmi` | Removes one or more images. | `docker rmi nginx` |
| `docker image prune` | Removes unused images. | `docker image prune` |
| `docker history` | Displays image layer history. | `docker history nginx` |
| `docker save` | Saves an image to a tar archive. | `docker save -o nginx.tar nginx` |
| `docker load` | Loads an image from a tar archive. | `docker load -i nginx.tar` |

---

## 3. Docker Containers

| Command | Description | Example |
|---------|-------------|---------|
| `docker run` | Creates and starts a container. | `docker run -d nginx` |
| `docker create` | Creates a container without starting it. | `docker create nginx` |
| `docker start` | Starts a stopped container. | `docker start web` |
| `docker stop` | Stops a running container. | `docker stop web` |
| `docker restart` | Restarts a container. | `docker restart web` |
| `docker pause` | Pauses a container. | `docker pause web` |
| `docker unpause` | Resumes a paused container. | `docker unpause web` |
| `docker kill` | Forcefully stops a container. | `docker kill web` |
| `docker rename` | Renames a container. | `docker rename web nginx-web` |
| `docker rm` | Removes a container. | `docker rm web` |
| `docker container prune` | Removes all stopped containers. | `docker container prune` |

---

## 4. Container Monitoring & Debugging

| Command | Description | Example |
|---------|-------------|---------|
| `docker ps` | Lists running containers. | `docker ps` |
| `docker ps -a` | Lists all containers. | `docker ps -a` |
| `docker logs` | Displays container logs. | `docker logs web` |
| `docker exec` | Runs a command inside a running container. | `docker exec -it web bash` |
| `docker inspect` | Displays detailed information about Docker objects. | `docker inspect web` |
| `docker top` | Displays running processes inside a container. | `docker top web` |
| `docker stats` | Displays live resource usage. | `docker stats` |
| `docker diff` | Shows file system changes inside a container. | `docker diff web` |
| `docker port` | Lists port mappings. | `docker port web` |

---

## 5. File Management

| Command | Description | Example |
|---------|-------------|---------|
| `docker cp` | Copies files between host and container. | `docker cp file.txt web:/tmp/` |

---

## 6. Docker Volumes

| Command | Description | Example |
|---------|-------------|---------|
| `docker volume create` | Creates a volume. | `docker volume create myvolume` |
| `docker volume ls` | Lists volumes. | `docker volume ls` |
| `docker volume inspect` | Displays volume details. | `docker volume inspect myvolume` |
| `docker volume rm` | Removes a volume. | `docker volume rm myvolume` |
| `docker volume prune` | Removes unused volumes. | `docker volume prune` |

---

## 7. Docker Networks

| Command | Description | Example |
|---------|-------------|---------|
| `docker network create` | Creates a network. | `docker network create bank-network` |
| `docker network ls` | Lists Docker networks. | `docker network ls` |
| `docker network inspect` | Displays network details. | `docker network inspect bridge` |
| `docker network connect` | Connects a container to a network. | `docker network connect bank-network web` |
| `docker network disconnect` | Disconnects a container from a network. | `docker network disconnect bank-network web` |
| `docker network rm` | Removes a network. | `docker network rm bank-network` |
| `docker network prune` | Removes unused networks. | `docker network prune` |

---

## 8. Docker Registry

| Command | Description | Example |
|---------|-------------|---------|
| `docker login` | Logs in to a Docker registry. | `docker login` |
| `docker logout` | Logs out of a Docker registry. | `docker logout` |
| `docker search` | Searches Docker Hub for images. | `docker search nginx` |

## 9. Docker Compose

| Command | Description | Example |
|---------|-------------|---------|
| `docker compose up` | Starts services from a Compose file. | `docker compose up -d` |
| `docker compose down` | Stops and removes services. | `docker compose down` |
| `docker compose ps` | Lists Compose containers. | `docker compose ps` |
| `docker compose logs` | Displays Compose logs. | `docker compose logs` |
| `docker compose restart` | Restarts Compose services. | `docker compose restart` |

## 10. Docker Cleanup

| Command | Description | Example |
|---------|-------------|---------|
| `docker system prune` | Removes unused Docker resources. | `docker system prune -a` |
| `docker image prune` | Removes unused images. | `docker image prune` |
| `docker container prune` | Removes stopped containers. | `docker container prune` |
| `docker volume prune` | Removes unused volumes. | `docker volume prune` |
| `docker network prune` | Removes unused networks. | `docker network prune` |
